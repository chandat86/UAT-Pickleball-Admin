📹 Camera trực tiếp các sân
▾
📌 Mỗi sân 1 điện thoại quay → stream lên YouTube Live hoặc dùng app IP Webcam → dán link vào đây để xem trực tiếp trong app.
📖 Hướng dẫn setup camera điện thoại
📡 IP Webcam (cùng WiFi): Cài app "IP Webcam" (Android) → Start server → nhập URL http://IP:8080
🎥 Jitsi Meet (qua internet): Vào meet.jit.si → tạo phòng (VD: PBDaNangSan1) → dán link vào đây → bấm "🔗 Mở tab mới"
▶️ YouTube Live: Nhập link stream YouTube trực tiếp
Cách 1 – YouTube Live (xem được từ xa, miễn phí):
1. Mở YouTube app → nhấn ＋ → Live
2. Chọn Stream ngay → copy link stream
3. Dán link YouTube vào ô trên
4. App sẽ embed video YouTube trực tiếp

Cách 2 – IP Webcam (chỉ cần WiFi nội bộ, độ trễ thấp):
1. Tải app IP Webcam (Android) hoặc EpocCam (iOS)
2. Mở app → nhấn Start server
3. App hiện URL dạng http://192.168.x.x:8080
4. Dán URL đó vào ô trên → chọn loại IP Webcam
5. Cả 2 điện thoại phải cùng WiFi
