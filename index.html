Chọn nội dung để xem BXH

📖 Hướng dẫn tạo giải & lịch thi đấu
Step-by-step cho admin
1Xác định nội dung & bảng đấu
Mỗi nội dung (VD: Đôi Nam 5.4, Đơn Nữ 4.0) có nhiều bảng (VD: Bảng A, B, C).
Mỗi bảng có nhiều trận. Xác định trước rồi mới tạo lịch.
2Thêm VĐV vào hệ thống
Cách nhanh: Vào Vận động viên → 📥 Import CSV để thêm hàng loạt.
Cách thủ công: Vào Vận động viên → ＋ Thêm VĐV từng người.
Điền đúng cột Nội dung để lọc VĐV theo nội dung thi đấu.
3Tạo lịch thi đấu
⚡ Round-Robin: Lịch thi đấu → Round-Robin → nhập danh sách đội → tự ghép cặp.
📥 Import CSV: Lịch thi đấu → Import lịch → tải mẫu → điền → upload.
📊 Google Sheets: Nhập thẳng vào sheet Matches (nhanh nhất khi có nhiều trận).
＋ Thủ công: Thêm từng trận qua form.
4Phân công trọng tài
Vào Trọng tài → Thêm/Sửa → điền Sân được phân và Vòng đấu.
Để trống = trọng tài thấy tất cả. Ngăn cách bằng dấu phẩy.
5Phát app cho trọng tài
Gửi file pickleball_referee_final.html cho từng trọng tài.
Trọng tài mở → nhập mã (REF001...) → chấm điểm.
📥 Cách import lịch thi đấu
📄 Import CSV (trên Admin Panel)
1. Vào Lịch thi đấu
2. Nhấn 📥 Import lịch
3. Tải file mẫu CSV
4. Điền dữ liệu vào file
5. Upload file → Import
📊 Nhập Google Sheets (nhanh nhất)
1. Mở Google Sheet → tab Matches
2. Nhập dữ liệu theo đúng cột
3. App tự đồng bộ khi làm mới
⭐ Dùng cách này khi có 50+ trận
📊 Format cột trong Google Sheets — tab Matches
Cột A: Nội dung (VD: Đôi Nam 5.4) Cột B: Bảng (VD: Bảng A) Cột C: Mã trận (VD: A1, A2, QF1...) Cột D: Đội A (VD: Nguyễn A - Trần B) Cột E: Điểm A (để trống khi chưa đấu) Cột F: Điểm B (để trống khi chưa đấu) Cột G: Đội B (VD: Lê C - Phạm D) Cột H: Sân (VD: Sân 1) Cột I: Vòng đấu (VD: Vòng bảng) Cột J: Ghi chú (để trống)
📄 Format CSV lịch thi đấu
noidung,bang,ma,doiA,doiB,san,vong,ghichu Đôi Nam 5.4,Bảng A,A1,Nguyễn A - Trần B,Lê C - Phạm D,Sân 1,Vòng bảng, Đôi Nam 5.4,Bảng A,A2,Hoàng E - Vũ F,Đặng G - Bùi H,Sân 2,Vòng bảng, Đôi Nam 5.4,Knock-out,QF1,TBD,TBD,Sân 1,Tứ kết,
📄 Format CSV vận động viên
id,name,gender,dob,club,phone,email,category,level,note P001,Nguyễn Văn An,Nam,1990-01-15,CLB Đà Nẵng,0901234001,,Đôi Nam 5.4,5.4, P002,Trần Văn Bình,Nam,1992-05-20,CLB Hội An,0901234002,,Đôi Nam 5.4,5.4, P003,Lê Thị Cúc,Nữ,1995-08-10,CLB Đà Nẵng,0901234003,,Đôi Nữ 4.0,4.0,
